# Grave Robber Punk

Static website for **graverobberpunk.com**.

## Local preview

```bash
npx serve . -l 8080
```

Open: http://localhost:8080

## Edit content

- Home page text: `index.html`
- Shows: `shows.js`
- Mailing list form: `signup.html`
- Marquee bands/songs: `graveyard.js`
- Theme/colors/layout: `styles.css`
- Domain for GitHub Pages: `CNAME`

## Deploy

This is a static site and can be deployed with GitHub Pages, Netlify, Vercel, or any standard web host.
